# Marketing Campaign Performance Analysis

## Project Overview
This project analyzes marketing campaign performance using Excel, SQL, Python, and Power BI.

The objective was to identify:
- High-performing marketing channels
- Profitable customer segments
- ROI trends
- Campaign effectiveness
- Business optimization opportunities

---

## Tools Used
- Excel
- MySQL
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

---

## Key Insights
- Facebook generated highest ROI
- Foodies and Fashionistas segments were most profitable
- ROI remained stable across months
- Interactive dashboard improved business decision-making


---

## Project Workflow

Raw Data → Excel Cleaning → SQL Analysis → Python Visualization → Power BI Dashboard
